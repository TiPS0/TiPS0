## 🚀 Featured Project — [`@p_tipso/agentive`](https://github.com/TiPS0/agentive)

**A universal, framework-agnostic AI agent workspace CLI**

[![npm version](https://img.shields.io/npm/v/@p_tipso/agentive?style=for-the-badge&color=7c3aed&logo=npm)](https://www.npmjs.com/package/@p_tipso/agentive)
[![npm downloads](https://img.shields.io/npm/dm/@p_tipso/agentive?style=for-the-badge&color=4f46e5&label=downloads)](https://www.npmjs.com/package/@p_tipso/agentive)
[![GitHub Stars](https://img.shields.io/github/stars/TiPS0/agentive?style=for-the-badge&color=7c3aed&logo=github)](https://github.com/TiPS0/agentive/stargazers)
[![License: MIT](https://img.shields.io/badge/License-MIT-a78bfa?style=for-the-badge)](https://opensource.org/licenses/MIT)

Stop maintaining separate rule files for every AI tool. **agentive** scaffolds a `.agents/` workspace, `AGENTS.md`, and `.aiignore` — a **single source of truth** for all your AI agents.

```bash
npx @p_tipso/agentive@latest
```
